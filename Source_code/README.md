# For Development
## Running

1.  🙂 Windows

```sh
# 🗻 Create environment
python -m venv env
.\env\Scripts\activate

# ⚙ Setup environment
pip install -r .\requirements.txt

# 🏃‍♂️ Run
python .\app.py 
```

2.  🐧 Linux

```sh
# 🗻 Create environment
python3 -m venv env
source env/bin/activate

# ⚙ Setup environment
pip3 install -r requirements.txt

# 🏃‍♂️ Run
python3 app.py
```
